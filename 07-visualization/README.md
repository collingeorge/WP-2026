## SENI v4 Global Intelligence Map

**[Open Map →](https://collingeorge.github.io/WP-2026/07-visualization/SENI_v4_Global_Map_FINAL.html)**

374 georeferenced nodes across 25 threat networks:

- Russia-China strategic infrastructure
- Russian nuclear fuel cycle (47 nodes)
- Iran WMD and missile program
- Makran Coast industrial complex
- Hormuz crisis architecture
- DPRK nuclear and cyber-financial networks
- Global cartel financial infrastructure
- Terrorist finance networks
- PRC and Russian overseas military basing
- Wagner Group / Africa Corps
- IRGC-Quds Force external operations
- BRI strategic infrastructure nodes

Satellite imagery via Esri World Imagery. All nodes sourced from open-source public record only. Confidence levels, legal authorities, and enforcement theory carried through per node. OSINT ceiling doctrine applies — see `05-seni-system/00-architecture/WP-2026-SENI-ARCH-01.pdf`.

## MDO-01 Analytic Console

**[Open Console →](https://collingeorge.github.io/WP-2026/07-visualization/WP-2026-MDO-01-console.html)**

Interactive decision-support interface operationalizing MDO-01 v3.0. Ten modules:

1. Three-track comparator (Diplomatic / Overt Warfare / Clandestine)
2. Authorities and friction map
3. Adversary stress test (PRC / Russia / DPRK)
4. Campaign workflow viewer
5. Confidence matrix
6. F2T2EA targeting sequence
7. Sanctions case validation
8. Scenario builder
9. Export brief generator
10. Reading guide with dependency links

The three-track framework is an analytic organizing construct derived from publicly documented authorities and observed practice. It is not a formally codified U.S. decision architecture.

## Series Position

Visualization artifacts sit at the top of the analytical stack — they operationalize and present findings from the full corpus. The SENI map draws from SENI-01 (`04-applied-enforcement`) and the full SENI sub-series (`05-seni-system`). The MDO-01 console operationalizes MDO-01 v3.0 doctrine. Neither artifact introduces new analytical claims — all claims are sourced from upstream series documents.
